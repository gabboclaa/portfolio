# Gabriele Clara Di Gioacchino — Portfolio

Personal portfolio website built with Next.js, TypeScript, and Tailwind CSS. Features a hero section, projects showcase, skills section, dark/light theme, and an inline CV viewer.

**Stack:** Next.js 16 · TypeScript · Tailwind CSS · React PDF
**Deployment:** Vercel

## Getting Started

```bash
git clone https://github.com/gabboclaa/portfolio
cd portfolio
npm install
npm run dev
```
